# Code Mastery
*Master code, one quiz at a time.*

## Project Overview
**Code Mastery** is a Django-based AI quiz platform where students can generate, take, and share quizzes. The platform supports:

- AI-generated quizzes from prompts  
- Random quizzes via the randome/dice button  
- Manual quiz creation (CRUD)  
- Public quiz sharing via unique links  
- Progress tracking: points, streaks, badges  
- Gamified dashboard for registered users  

> Currently in early development.
